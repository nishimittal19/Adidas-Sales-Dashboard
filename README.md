# Adidas Sales Performance Dashboard
Project Overview
This interactive Power BI dashboard provides a comprehensive analysis of Adidas Sales Data, focusing on key performance indicators (KPIs) like total revenue, profitability, and unit sales across different regions and product categories. The goal is to identify high-performing products and regional trends to support data-driven business decisions.

Key Features & Insights
The dashboard is divided into several analytical sections:

Executive Summary (KPIs):

Total Sales: Highlighting a massive $899.90M in overall revenue.

Total Profit: An impressive $332.1M sum of operating profit.

Average Price Per Unit: Tracking pricing trends, currently at $45.2.

Total Units Sold: Real-time tracking of inventory movement (currently showing 2M units in the master view).

Product Performance:

A horizontal bar chart ranking products by sales. Men's Street Footwear and Women's Apparel are the leading categories.

Dynamic filtering allows users to see individual product contributions to total revenue.

Geographic Analysis:

Regional Distribution: A donut chart showing sales split across regions (West, Northeast, Southeast, etc.), with the West region leading at 30% of total sales.

Spatial Mapping: An interactive map visualizing sales density across different states/locations (specifically focusing on South Africa and global clusters in the provided views).

Interactive Slicers:

The dashboard supports cross-filtering; clicking on a specific region or product category updates all other visuals to reflect that specific data subset.

Tools Used
Power BI Desktop: For data modeling and visualization.

DAX (Data Analysis Expressions): Used for creating custom measures like Total Profit and Average Price.

Power Query: For data cleaning and transformation.

How to View
Download the .pbix file from this repository.

Open it using Power BI Desktop.

Interact with the charts to explore different data dimensions.
